时光

发表于 2017-08-18 | 分类于 诗 | 阅读次数:

你是否

还能够再次想起

花瓣铺满纯白的画面

你轻抚着我的脸

擦去我感动的泪

说要一起看逝水流年

回忆中

呢喃依稀在耳畔

咿呀学语可爱的模样

岁月将你从怀里

送往未知的明天

那个微笑竟是永远

相册里

那些褪色的画

斑驳的过往

很多没说完的话

脑海中

怀念的冲动不停的翻涌

流动着牵挂

怎能遗忘

我想抓住时光

岁月的泪却沉在了眼眶

顺着心的方向

散开的回忆铺成了网

多想抓住时光

留住所有美好的过往

只愿记忆之中

独留你的陪伴

谢谢请俺吃糖
