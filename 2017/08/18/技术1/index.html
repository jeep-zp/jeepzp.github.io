hexo 主页文章数翻页设置

发表于 2017-08-18 | 分类于 技术 | 阅读次数:

前面查看了很多的资料也没有在网上看见关于hexo设置翻页的教程，后来折腾了半天才发现，原来hexo自己已经可以实现翻页，不需要外部插件的支持。于是自己开始写一个简单操作的 教程。 —— 效果

第一步

首先查看自己hexo的版本。
在自己博客所在文件夹右键，选择Git Base Here

code

显示出版本信息。

第二步

到hexo 的config.yml中找到：

path: 指的是你设置的页面，主页是默认选项。
per_page:指的是你设置的需要显示的文章数。
order_by:指的是按时间的排序。

第三步

hexo clean
hexo g
hexo s
运行后查看效果。
